# Premium Full-Stack Developer Portfolio

A high-performance, recruiter-ready developer portfolio built with Next.js, TypeScript, and Framer Motion. This project showcases a luxury design aesthetic with premium typography, glassmorphism, and interactive AI-powered component engineering.

## ✨ Key Features

- **Luxury Dark Theme**: A sophisticated custom color palette with gold and white accents.
- **Dynamic Tech Orbit**: A responsive, animated planet-style technology showcase.
- **Interactive Project Cards**: Premium glassmorphism cards featuring problem/solution breakdowns and technical deep-dives.
- **Engineering Concepts**: Specialized section for highlighting architectural knowledge and core computer science fundamentals.
- **Glassmorphism UI**: Modern, translucent layout components built with Tailwind CSS.
- **Fully Responsive**: Optimized for all devices, from mobile to desktop.
- **SEO Optimized**: Built-in accessibility and search engine optimization.

## 🛠️ Technology Stack

- **Frontend**: [Next.js 14](https://nextjs.org/), [React](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ansh35/PORTFOLIO.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `src/app`: Next.js App Router and page layouts.
- `src/components`: UI components organized by sections and layouts.
- `src/data`: Centralized portfolio data (projects, skills, experience).
- `public`: Static assets including images and documentation.
